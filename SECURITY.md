# Security Policy

## Reporting a vulnerability

This is a public quickstart repository that demonstrates zero-auth API usage. It does not process user data or handle credentials. If you discover a vulnerability in this repo (e.g. malicious dependency, workflow injection, supply-chain issue), please report it privately:

- Email: security@apier.no
- Or via the apier.no responsible-disclosure page: https://apier.no/security

For vulnerabilities in the underlying apier.no platform itself, use the same channels above.

We aim to acknowledge reports within 72 hours.
